Syncra AI 
Ecosistema de Ventas Web y Móvil con Integración de Gemini AI

Descripción del Proyecto: Syncra AI es una solución CRM integral diseñada para Pequeñas y Medianas Empresas (PYMES) en Guatemala. Unifica el proceso de ventas conectando a los gerentes en la oficina (vía Dashboard Web) con los agentes de campo (vía App Móvil Nativa), 
asegurando que ningún cliente potencial se pierda por falta de seguimiento.

Características Clave: 
* Dashboard Web: Tablero estilo Kanban para gestionar etapas de ventas (Next.js).
* App Móvil: Aplicación nativa Android para que los agentes registren visitas y contactos (Kotlin).
* Integración IA (Gemini): "Generador de Discursos Inteligente" (Smart Pitch) que crea guiones de venta personalizados basados en el historial del cliente.
* Sincronización en Tiempo Real: Impulsado por Firebase Firestore.

-------------------------------

Stack Tecnológico: 
* Frontend Web: Next.js + Tailwind CSS.
* App Móvil: Android Nativo (Kotlin).
* Backend/Auth: Firebase (Auth, Firestore, Cloud Functions).
* Inteligencia Artificial: Google Gemini API.
